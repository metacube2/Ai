- Produktsparten-Mapping ist lokal/committed auf den neuen SAP-OData-Referenzservice vorbereitet.
- `P = ProductDivisionRefSet` bleibt Alias und fuehrende Quelle.

- Alter Join `Z.Prodh = M.Paph1` ist inaktiv.
- Produktfelder kommen direkt aus `P.*`:
  - `P.Paph1`, `P.Paph1Text`
  - `P.Wwpfa`, `P.WwpfaText`
  - `P.Wwpsp`, `P.WwpspText`
  - `P.IsAssigned`
- OData-Join normalisiert `Matnr` auf beiden Seiten:
  - Trim
  - ToUpperInvariant
  - Whitespace entfernen
  - fuehrende Nullen entfernen
- Neue Kategorie `Uebrige` fuer `ProductDivisionCode = 0008`; getrennt von `Nicht zugeordnet` und `Nicht im TR-AG-Stamm`.
- Guardrails:
  - grosse ProductDivisionRefSet-Referenz mit 0 assigned bricht Import ab.
  - SAP-Fetch mit 0 Sales Records bricht ab, damit CentralSalesRecords nicht leer ueberschrieben wird.
  - SAP Gateway Timeout auf 5 Minuten erhoeht.
